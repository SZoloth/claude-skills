## Coaching Session Structure

**Weekly 1-on-1 Format** (55 minutes):
- **Opening** (5 min): Wins, learnings, morale check
- **Discovery Review** (20 min): Latest work, challenge assumptions
- **Coaching Moment** (15 min): Teach one specific skill
- **Practice/Application** (10 min): Apply to real work
- **Next Steps** (5 min): Commitments, obstacles

**Monthly Review Format** (90 minutes):
- Assessment update (people, process, product)
- Coaching plan adjustment
- Skill demonstration
- Goal setting for next month
